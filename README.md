# New Fancy Dresses E-Commerce Platform

A production-ready Next.js e-commerce application for the New Fancy Dresses clothing store.

## Features

- **Category Navigation**: Flipkart-style filtering
- **Advanced Filtering**: Brand filters and price range selectors
- **Smart Search**: Real-time search with suggestions
- **Product Showcase**: Image carousel with size charts
- **Shopping Cart**: Full management with promo codes
- **Secure Checkout**: Address validation
- **Admin Dashboard**: Product management
- **Reviews & Ratings**: Customer feedback
- **PDF Invoices**: Auto-generated invoices
- **Multi-Language**: English and Bangla
- **Mobile Optimized**: Responsive design

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Zustand
- jsPDF
